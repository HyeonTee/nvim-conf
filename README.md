## Frequently Using Shortcuts (추가 예정)
- s + v : 화면 수직 분할
- s + s : 화면 수평 분할
- s + h,j,k,l : 분할 화면 이동
- space + e : neotree 토글
- shift + h,l : Buffer 이동
- g + c : 주석 처리 (with visual mode)
- g + d : go to definition
- space + l : lazyvim
